# 嵌入式入门认知

## 嵌入式定义
·以应用为中心，以计算机技术为基础，软硬件可裁剪，适应应用系统对功能、可靠性、成本、体积、功耗等严格要求的专用计算机系统  
·不是为运行各种软件设计的，而是为某个特定任务或一类任务服务的  
·本质上是一台计算机，包含处理器、存储器等核心部件  
·根据实际需求，设计人员可以对系统的硬件配置和软件功能进行精简或扩展  

## 系统组成
·嵌入式处理器：负责执行指令，如单片机(MCU)，微处理器  
·外围硬件设备：存储器以及各种接口，用于连接传感器、显示屏、电机等外部设备  
·嵌入式操作系统  
·用户应用程序：实现系统特定功能的软件代码  

## 应用场景
·消费电子：智能手机、智能手表、数码相机、智能家电（空调、洗衣机）     
·汽车电子：发动机控制单元（ECU）、防抱死制动系统（ABS）、车载娱乐系统       
·工业控制：机器人、自动化生产线控制器、数据采集设备    
·医疗设备：心电图机、血糖仪、B超设备    
·网络通信：路由器、交换机、机顶盒   

## MCU方向
围绕微控制器进行的软硬件开发  
### 工作内容
·驱动开发  
·协议解析：处理各种通信协议，如蓝牙，WiFi  
·逻辑控制：编写业务代码  
### 技术栈
·入门基础：C语言、数字/模拟电路  
·硬件交互：GPIO、中断、定时器  
·通信接口：UART, I2C, SPI, ADC  
·进阶内核：RTOS (FreeRTOS/RT-Thread) 
·行业工具：Keil, IAR, VSCode, Git  
·调试手段：示波器、逻辑分析仪  
### MCU未来
MCU+AI（边缘AI）：现在的MCU开始集成NPU（神经网络单元），能在本地识别人脸、语音或检测设备故障，不再依赖云端
